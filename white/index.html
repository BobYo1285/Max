Белый экран Maх

Эта страница предоставляет чистый белый экран. В режиме приложения (после добавления на главный экран) будет отображаться только белый фон.

Как установить на iPhone:

  1. Нажмите кнопку "Поделиться" внизу браузера Safari
  2. Прокрутите вниз и выберите "На экран «Домой»"
  3. Нажмите "Добавить" в правом верхнем углу
  4. Готово! Теперь приложение будет на вашем домашнем экране
